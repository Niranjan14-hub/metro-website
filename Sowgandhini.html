How it works — pipeline overview

  1. Ingest telemetry, AFC (ticketing), passenger counts, and maintenance windows.
  2. Short-term demand forecasting (X minutes/hours ahead) using ML models.
  3. Optimization engine (MILP / heuristic / metaheuristic) computes induction assignments subject to constraints.
  4. Planner suggests schedule & re-optimization runs in real-time as events occur.
  5. Operator dashboard and API endpoints allow human-in-the-loop overrides & auditing.
